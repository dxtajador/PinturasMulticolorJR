Baslac

BASLAC Base Color es un Sistema Acrílico diseñado con resinas de alto desempeño es de fácil uso, con buen poder cubriente y de secado rápido, este material es de alta calidad ya que se utiliza un Sistema Bicapa para recubrirla dándole así protección y durabilidad. Se caracteriza también por tener un rendimiento mayor a cualquier recubrimiento.

Categorías: Línea Automotriz, Repintado Automotriz
